# 🎙️ AI Voice Assistant

A real-time multilingual AI Voice Assistant built using Flask, JavaScript, Gemini AI, and ElevenLabs Text-to-Speech.

The assistant can understand user input, generate intelligent AI responses, and reply using realistic voice output.

---

# 🚀 Features

* 🎤 Real-time Voice Assistant
* 💬 AI Chat Response Generation
* 🔊 Human-like Voice Output
* 🌍 Multi-language Support
* 🧠 Gemini AI Integration
* 🎵 ElevenLabs Text-to-Speech
* 🖥️ Interactive Frontend UI
* ⚡ Flask Backend API
* 📱 Responsive Design

#  Installation

## Clone Repository

git clone <YOUR_GITHUB_REPO>

## Create Virtual Environment

py -3.10 -m venv venv

## Activate Virtual Environment

venv\Scripts\activate

## Install Dependencies

pip install -r requirements.txt

## Run Project

python app.py
